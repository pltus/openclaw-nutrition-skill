# Privacy & Security Notes

- Default behavior is fully local file I/O only.
- No network calls are required for text-only logging or summaries.
- Image-based nutrition estimation may use a connected model/network only after explicit user consent.
- Do not store API keys, secrets, or credentials in repo files.
- Keep sensitive user data inside `data/` and outside version control.
- Shell commands, downloads, or external lookups should only happen with explicit user confirmation.
